# **App Name**: ColonyGo Sales Navigator

## Core Features:

- Opportunity Management: Create, update, and manage sales opportunities, linking them to clients, products, and sales stages.
- Action Tracking: Track actions related to opportunities with status indicators and due dates, integrated with the ColonyGo ticketing system.
- AI-Powered Alerts: Receive automated alerts generated by GyneAI for potential issues such as delays, anomalies, or risks associated with opportunities. The LLM uses reasoning as a tool to trigger those alerts.
- News Feed: Provide a real-time news feed of updates to subscribed opportunities, showing changes in status, new quotes, invoices, and notes.
- Sales Data Visualization: Visualize the sales pipeline, won vs lost opportunities, client profiles, and workload distribution using charts and graphs.
- Meeting Notes: Capture, create, update and maintain meeting notes.
- AI Assistance: Enable users to interact with GyneAI via text, voice, or file uploads to automate tasks such as report creation, task generation, and document processing. An IA icon provides provenance.

## Style Guidelines:

- Primary color: Navy blue (#008575) for titles, icons, and bars, conveying professionalism.
- Background color: Very light gray (#F8FAFC) for the page background, ensuring a clean and airy feel.
- Accent color: Light blue (#2095C6) for links and informational elements, adding a touch of clarity.
- Headline font: 'Poppins' (sans-serif) for titles, providing a modern and clear appearance.
- Body font: 'Inter' (sans-serif) for the body text, offering a readable and contemporary style.
- Use simple, outlined icons in navy blue or light blue to represent different actions and statuses.
- Maintain a maximum width of 1200px for the main content area.
